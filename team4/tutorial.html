Getting Started

npm install

Simply run npm install team-a-icomponent, you should be able to find a folder called "team-a-icomponent" in the "node_module" folder under the same directory where you ran the command.

To add our components to your page, copy the "team_a_icomponent" folder to where your html file is located, and include <script src="team-a-icomponent/button/button-component.js"></script> in your html file.
