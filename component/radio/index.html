Radio Button

Basic Usage

Basic radio button with multiple options

    <radio-component name="gender">
        <radio-option value="male" label="Male"></radio-option>
        <radio-option value="female" label="Female"></radio-option>
        <radio-option value="other" label="Other"></radio-option>
        <radio-option value="disabled" label="Disabled" disabled></radio-option>
    </radio-component>

With borders

The border attribute adds a border to the options

    <radio-component name="color">
        <radio-option border value="red" label="Red"></radio-option>
        <radio-option border value="yellow" label="Yellow"></radio-option>
        <radio-option border value="green" label="Green"></radio-option>
        <radio-option border value="other" label="Other" disabled></radio-option>
    </radio-component>
