Slider

Basic Usage

Default min is 0 and max is 100

<slider-component></slider-component>

Custom min/max

Can also set the default starting value using value attribute

<slider-component value="2015" min="2010" max="2020"></slider-component>

Support for bootstrap theme

Set the color of the slider by setting type attribute






<slider-component value="5" min="0" max="30" type="secondary"></slider-component>
<slider-component value="10" min="0" max="30" type="success"></slider-component>
<slider-component value="15" min="0" max="30" type="info"></slider-component>
<slider-component value="20" min="0" max="30" ype="primary"></slider-component>
<slider-component value="25" min="0" max="30" type="warning"></slider-component>
<slider-component value="30" min="0" max="30" type="danger"></slider-component>
